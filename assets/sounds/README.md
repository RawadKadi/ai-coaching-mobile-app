# Notification Sound

A notification sound file should be placed here.

## Required File:
- `notification.mp3` (or .wav)

## Recommended:
- Short duration (0.5-1 second)
- Pleasant, non-intrusive tone
- Light notification sound

## Free Sources:
- https://notificationsounds.com/
- https://freesound.org/
- iOS/Android system sounds

For now, you can download a free notification sound and place it here as `notification.mp3`.

If the file is missing, the app will still work but won't play sounds.
